# Where I Open
> [[멀티캠퍼스 데이터 분석&엔지니어 26회차] 참 잘했조(1조) 공유 레포지토리]

## 주제
시간단위별 유동인구 데이터를 활용한 지역별 매출 예측
- 부제 : 특정 업종의 매출 및 유동인구 데이터를 이용한 최적의 자리 찾기 

## 요약 설명
시간대별 유동인구 데이터를 통하여 자치구 및 동 단위의 매출 분석 및 예측

## 조원
조장 : 라영채
조원 : 곽현아, 노희연, 윤제욱, 정혜원

## 폴더
- Pre_processing : 전처리 및 데이터 가공 프로세스 관련 프로그램
